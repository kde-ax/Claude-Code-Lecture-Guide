03 · Prerequisites

시작 전 준비물

세 가지만 확인하세요. 도구 설치는 다음 장부터.

💻

Windows 환경

  • · Windows 10 (1809+) / 11 권장
  • · RAM 8GB 이상 권장 (최소 4GB)
  • · x64 또는 ARM64 CPU
  • · 디스크 여유 1~2GB 이상
  • · 인터넷 연결 필수
🔑

Anthropic 유료 계정

  • · Claude Pro · Max · Team · Enterprise
  • · 또는 Anthropic Console API
  • · 무료 Claude.ai 플랜만 사용 불가
⌨️

PowerShell (PS 5.1 기본 탑재)

  • · Windows에 이미 포함 · 설치 불필요
  • · 시작 메뉴 → "PowerShell" 검색 → 실행
  • · 관리자 권한 불필요
  • · CMD 아님 · 주의

04 · Installation Checklist

설치 체크리스트

필수 먼저, 권장 그 다음, 선택은 마지막. 순서대로 진행하면 충돌 없습니다.

필수 · Required

1 · Git for Windows

Claude Code가 내부적으로 Git Bash 호출. 없으면 실행 안 됨.

2 · Claude Code

본체. PowerShell 한 줄 설치. npm 방식 아님.

권장 · Recommended

3 · Node.js (v24 LTS)

Claude Code 자체엔 비필수. 향후 MCP 연동·AI 도구에 사용.

4 · VS Code

Claude가 수정한 파일 확인·편집. 내장 터미널도 편리.

선택 · Optional

5 · Claude Desktop

Claude를 GUI 앱으로 사용 가능. Claude Code도 데스크톱 앱 안에서 실행 가능(터미널이 어려운 분에게).

Git for Windows — 가장 먼저 설치

Claude Code 실행에 필요한 기본 환경을 만들어줍니다.

Step 01 · 다운로드

Step 02 · 설치 · 3가지 확인
  • ✅ 바탕화면 아이콘
  • ✅ Windows 터미널에 Git Bash 추가
  • ✅ 기본 편집기 → VS Code
Step 03 · 설치 확인
# PowerShell
git --version
→ 2.46.x.windows.1

Step 02 · 기본 편집기 (VS Code)

Claude Code 설치 — PowerShell 한 줄

PowerShell에 한 줄 붙여넣기.

# 1. PowerShell 열기 (관리자 권한 X)
# 시작 메뉴 → "PowerShell" 검색 → 클릭

# 2. 아래 한 줄 복사해서 붙여넣기 + 엔터
irm https://claude.ai/install.ps1 | iex

✻ Downloading Claude Code…
✓ Installed to %USERPROFILE%\.local\bin

# 3. PowerShell 닫았다가 다시 열기 (PATH 적용)
# 4. 설치 확인
claude --version
2.1.x

왜 이 방법으로 설치하나요?

Node.js 설치 불필요

예전엔 별도 프로그램을 먼저 설치해야 했지만, 지금은 한 번에 설치돼요.

PowerShell에서만 실행 가능

irm 설치 명령어가 PowerShell 전용이라 다른 터미널에서 입력하면 오류가 나요.

자동 업데이트

설치 후 별도로 업데이트하지 않아도 최신 버전이 유지돼요.

Node.js v24.15.0 LTS 설치

향후 MCP 연동·AI 도구 사용을 위해 미리 깔아두면 편합니다.

Step 01 · 사이트 접속

nodejs.org → 초록색 "Get Node.js®" 버튼 클릭.

Step 02 · LTS 버전 확인

상단 드롭다운에 v24.15.0 LTS 선택돼 있는지 확인 (안정 버전).

Step 03 · 설치 프로그램 실행

초록색 "Windows 설치 프로그램(.msi)" 버튼 클릭 → 다운로드 후 실행 → 기본 옵션 Next → Install.

08 · Install · VS Code · 권장

VS Code — 코드 편집기 + 내장 터미널

Claude가 수정한 파일을 바로 확인·편집. VS Code 안에서 claude 실행도 가능.

Step 01 · 사이트 접속

code.visualstudio.com 접속.

Step 02 · 다운로드

검정색 "Download for Windows" 버튼 클릭 → 설치 파일 받기.

Step 03 · 설치 실행

받은 .exe 실행 → 라이선스 동의 → 기본 옵션 Next → Install.

Step 04 · (선택) Claude Code 확장

VS Code 실행 → Extensions (Ctrl+Shift+X) → "Claude Code" 검색 → Install. 에디터 안에서 Claude 바로 호출 가능.

VS Code 공식 홈
VS Code + Claude 터미널 연동
💡 VS Code 안에서 Ctrl + ` 로 터미널 열고 claude 실행.

09 · First Run & Login

첫 실행 & 로그인

여기까지 성공하면 CLI 환경 구축 완료. 다음 장부터 기본 조작.

# 1. PowerShell에서 Claude 실행
claude

# 2. claude 안에서 로그인
/login
→ 브라우저 자동 열림 → 승인
→ ✓ 자격 증명 자동 저장

# 3. claude 종료
/exit

# 4. ⚠️ PowerShell 에서 점검
# (claude 대화창 아님!)
claude doctor

순서 요약

1. claude 실행

PowerShell에서 claude. 환영 화면 확인.

2. /login · 브라우저 자동

기본 브라우저가 자동으로 Anthropic OAuth 페이지를 엽니다. 로그인 + 권한 승인.

3. 자격 증명 자동 저장

Windows Credential Manager에 저장 → 이후 자동 로그인.

4. ⚠️ claude doctor는 PowerShell

claude 대화창 안이 아닌, 종료 후 PowerShell 창에서 실행. 환경 점검 리포트가 나옵니다.

11 · In-Session Commands

자주 쓰는 명령어 모아보기

외울 필요 없어요. 막힐 때 꺼내 보세요.

💻

PowerShell 에서 — claude …

claude 실행 전·후, PowerShell 창에서 사용.

  • claude — 대화 시작
  • claude -c — 직전 세션 이어가기
  • claude --resume — 이전 세션 목록에서 선택
  • claude doctor — 환경 점검 리포트
  • claude update — 최신 버전 업데이트
  • claude --version — 버전 확인
  • claude -h — 도움말 전체
💬

claude 대화창에서 — / 명령어

/ 입력하면 전체 목록이 자동 표시.

  • /help — 명령 목록
  • /login — 로그인
  • /clear — 대화 초기화
  • /compact — 요약·토큰 절감
  • /init — CLAUDE.md 생성
  • /memory — 규칙 편집
  • /model — 모델 전환
  • /status — 상태 확인
  • /permissions — 권한 관리
  • /config — 설정
  • /cost — 사용량·비용
  • /exit — 종료
