## 정의

신경망은 **인간 뇌의 뉴런 구조에서 영감을 받은** 수학 모델로, 입력을 받아 여러 노드(뉴런)를 거쳐 출력을 만드는 계산 그래프다 [[20260504_AWS-LLM이란]].

각 노드는 입력값에 **가중치(weight)**를 곱하고 **편향(bias)**을 더해 활성화 함수에 통과시킨다. 가중치와 편향, 그리고 [[워드임베딩]]을 통틀어 [[파라미터]]라고 부른다.

## 핵심 구조

```
[입력층] → [은닉층 1] → [은닉층 2] → ... → [출력층]
```

- 한 층의 모든 노드는 다음 층의 모든 노드와 연결된다 (완전 연결, fully connected)
- 각 연결에 가중치가 매겨져 있다
- 학습은 정답과 출력의 오차를 줄이는 방향으로 이 가중치들을 조정하는 과정 (역전파, backpropagation)

층이 많으면(깊으면) [[딥러닝]], 적으면 얕은(shallow) 신경망이라 부른다.

## 일상 비유

수많은 직원이 층층이 일하는 회사라고 하자.
- **1층 직원**: 외부에서 들어온 원자료를 받아 약간 가공한 보고서를 작성
- **2층 직원**: 1층의 보고서들을 종합해 더 추상적인 분석 작성
- **3층 직원**: 2층 분석들을 모아 의사결정안을 작성
- **CEO (출력층)**: 최종 결정

각 직원은 누구의 보고서를 얼마나 신뢰할지(=가중치)를 가지고 있고, 회사가 잘못된 결정을 내리면 신뢰도(가중치)를 조금씩 조정한다. 이 과정을 수백만 번 반복하면 회사가 점점 더 정확한 결정을 내리게 된다.

## 신경망 종류

| 종류 | 특징 | 주 용도 |
|---|---|---|
| MLP (Multi-Layer Perceptron) | 가장 기본적인 완전연결 신경망 | 분류, 회귀 |
| CNN (Convolutional Neural Network) | 공간 패턴에 강함 | 이미지 처리 |
| [[RNN]] | 시퀀스 처리, 메모리 기능 | (구) 번역, 시계열 |
| [[트랜스포머]] | 셀프어텐션 + 병렬 처리 | [[LLM]], 현대 AI 표준 |

## [[LLM]]과의 관계

[[LLM]]은 거대한 [[트랜스포머]] 신경망이다. 수십억~수천억 개의 [[파라미터]]를 가진 신경망이고, 다음 [[토큰]] 예측 작업으로 학습된다.

## 관련 개념

- 상위: [[딥러닝]]
- 부품: [[파라미터]] (가중치 + 편향), [[워드임베딩]]
- 종류: [[트랜스포머]], [[RNN]]
- 응용: [[LLM]], [[생성형AI]]
