## 정의

딥러닝은 **여러 층(layer)으로 깊게 쌓인 [[신경망]]**을 사용해 데이터에서 복잡한 패턴을 자동으로 학습하는 머신러닝의 한 갈래다 [[20260504_AWS-LLM이란]]. "deep"은 신경망의 층이 많다는 뜻.

## 핵심 메커니즘

전통적 머신러닝은 사람이 "어떤 특징(feature)을 볼지" 직접 설계해야 했다. 딥러닝은 그 작업을 **신경망이 층을 거치며 스스로 학습**한다.

- **얕은 층**: 단순한 패턴 (예: 이미지의 가장자리, 색깔)
- **중간 층**: 조합된 패턴 (예: 눈, 코, 입의 형태)
- **깊은 층**: 추상적 개념 (예: "이건 고양이 얼굴이다")

층을 거칠수록 점점 더 추상적인 특징이 자동으로 만들어진다.

## 일상 비유

요리사가 처음 보는 음식을 분석한다고 하자.
- **1층**: "단맛, 신맛, 짠맛이 있군"
- **2층**: "이 조합은 토마토 베이스인 것 같다"
- **3층**: "허브 향이 있고 면이 들어갔다"
- **4층**: "이건 이탈리아 파스타다"

각 층은 아래 층의 결과를 입력 받아 더 추상적인 판단을 내린다. 딥러닝의 계층 구조도 이와 같다.

## 머신러닝과의 관계

```
인공지능 (AI)
└─ 머신러닝 (ML)
    └─ 딥러닝
        └─ 트랜스포머·CNN·RNN 등 신경망 아키텍처들
            └─ LLM
```

딥러닝은 ML의 한 종류고, [[LLM]]은 딥러닝의 한 응용이다.

## 응용

- **컴퓨터 비전**: 이미지 분류, 얼굴 인식, 자율주행 (CNN 기반)
- **자연어 처리**: 번역, 챗봇, [[LLM]] (트랜스포머 기반)
- **음성 인식**: Siri, Alexa의 음성 → 텍스트 변환
- **추천 시스템**: 넷플릭스, 유튜브의 콘텐츠 추천
- **생성형 AI**: 이미지·음악·텍스트 생성 ([[생성형AI]])

## 왜 지금 폭발했나

딥러닝 이론은 1980년대부터 있었지만 다음 3가지가 모이며 2010년대에 폭발했다:
1. **데이터 폭증** — 인터넷에서 학습 데이터 확보 가능
2. **GPU의 등장** — 병렬 연산으로 훈련 시간 단축
3. **알고리즘 개선** — 역전파(backpropagation) 최적화, 새 아키텍처([[트랜스포머]] 등)
